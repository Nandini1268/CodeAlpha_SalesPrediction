## Task 4 — Sales Prediction using Python

### What is this project?
This project predicts future Sales based on advertising spend across 3 channels — TV, Radio and Newspaper — using Machine Learning regression models.

### What I did:
- Loaded and explored the Advertising dataset with 200 rows
- Visualised the relationship between each ad channel and sales
- Built a correlation heatmap to find the most impactful channel
- Trained Linear Regression and Random Forest Regressor models
- Evaluated models using R2 Score, MAE and RMSE
- Identified which advertising channel has the highest impact on sales

### Results:
- Linear Regression R2 Score: around 0.90
- Random Forest R2 Score: around 0.97
- Random Forest performed better than Linear Regression

### Key Insights:
- TV has the highest impact on sales with a correlation of 0.78
- Radio is the second most effective channel with a correlation of 0.58
- Newspaper has the least impact on sales with a correlation of 0.23
- Business recommendation: Invest more in TV and Radio advertising for maximum sales growth

### Tools Used:
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

### Dataset:
Advertising Dataset — Kaggle (https://www.kaggle.com/datasets/bumba5341/advertisingcsv)

- task4_sales_prediction.ipynb
- Advertising.csv
